# XarxesLabs - Laboratorios de Redes y Videojuegos Online

Repositorio que almacena todas las prácticas de laboratorio de la asignatura de **Redes y Videojuegos Online** de la carrera de **Diseño y Desarrollo de Videojuegos** en **UPC CITM Terrassa**.

## 📋 Descripción

Este repositorio contiene los materiales, código fuente, documentación y recursos necesarios para las prácticas de laboratorio de la asignatura de Redes y Videojuegos Online. Está organizado de manera que facilite el acceso y la gestión de los diferentes módulos y prácticas.

## 🗂️ Estructura del Repositorio

```
XarxesLabs/
├── README.md                   # Este archivo
├── LICENSE                     # Licencia MIT
├── .gitignore                 # Archivos a ignorar
├── docs/                      # Documentación general
│   ├── guias/                 # Guías de instalación y configuración
│   └── recursos/              # Recursos adicionales
├── laboratorios/              # Prácticas de laboratorio organizadas por módulos
│   ├── modulo-01-fundamentos/ # Fundamentos de redes
│   ├── modulo-02-protocolos/  # Protocolos de comunicación
│   ├── modulo-03-sockets/     # Programación con sockets
│   ├── modulo-04-unity/       # Redes en Unity
│   └── modulo-05-multijugador/ # Sistemas multijugador
└── plantillas/               # Plantillas para nuevas prácticas
    ├── proyecto-base/
    └── documentacion/
```

## 🎯 Objetivos de Aprendizaje

- **Fundamentos de redes**: Comprender los conceptos básicos de comunicación en red
- **Protocolos de comunicación**: TCP/UDP, HTTP, WebSockets
- **Programación de sockets**: Implementación de cliente-servidor
- **Redes en Unity**: Integración de funcionalidades de red en videojuegos
- **Sistemas multijugador**: Desarrollo de juegos multijugador online

## 🚀 Uso del Repositorio

### Para Estudiantes

1. Clona el repositorio:
   ```bash
   git clone https://github.com/MartaGnarta/XarxesLabs.git
   ```

2. Navega al laboratorio correspondiente:
   ```bash
   cd XarxesLabs/laboratorios/modulo-XX-nombre/
   ```

3. Sigue las instrucciones específicas de cada práctica en su archivo `README.md`

### Para Profesores

- Cada módulo contiene una estructura consistente para facilitar la evaluación
- Los materiales están organizados por dificultad progresiva
- Incluye plantillas para crear nuevas prácticas

## 📚 Requisitos Técnicos

- **Unity**: Versión 2022.3 LTS o superior
- **Visual Studio** o **Visual Studio Code**
- **Git** para control de versiones
- **Conocimientos básicos** de C# y programación orientada a objetos

## 🤝 Contribuciones
